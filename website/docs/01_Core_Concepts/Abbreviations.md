# Abbreviations & Acronyms

This page contains common abbreviations and acronyms used throughout the documentation.

## Technology Abbreviations

### A

- **API**: Application Programming Interface
- **AWS**: Amazon Web Services

### B

- **BaaS**: Backend as a Service

### C

- **CI/CD**: Continuous Integration/Continuous Deployment
- **CSS**: Cascading Style Sheets

### D

- **DOM**: Document Object Model
- **Docusaurus**: Static site generator for documentation

### F

- **Firebase**: Google's mobile and web application development platform

### G

- **Git**: Distributed version control system
- **GitHub**: Web-based hosting service for Git repositories

### H

- **HTML**: HyperText Markup Language

### J

- **JavaScript**: Programming language for web development
- **JSON**: JavaScript Object Notation

### N

- **Node.js**: JavaScript runtime environment
- **npm**: Node Package Manager

### P

- **PWA**: Progressive Web App

### R

- **React**: JavaScript library for building user interfaces
- **REST**: Representational State Transfer

### S

- **SQL**: Structured Query Language
- **SSR**: Server-Side Rendering

### T

- **TypeScript**: Typed superset of JavaScript

### V

- **VS Code**: Visual Studio Code editor

## Development Terms

- **Build**: Process of converting source code into executable software
- **Deploy**: Process of making software available for use
- **Lint**: Process of checking code for potential errors and style issues
- **Refactor**: Process of restructuring code without changing its functionality

## Version Control

- **Branch**: Independent line of development in Git
- **Commit**: Snapshot of changes in Git
- **Merge**: Combining changes from different branches
- **Pull Request**: Request to merge changes from one branch to another
- **Repository**: Storage location for software packages

## Cloud Computing

- **IaaS**: Infrastructure as a Service
- **PaaS**: Platform as a Service
- **SaaS**: Software as a Service

_This list will be updated as new terms are introduced in the documentation._
